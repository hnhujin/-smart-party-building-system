 # 智慧党建系统

## 介绍
    响应国家“学习**”号召，基于大数据分析平台打造“互联网+”工作模式，以智慧党建为引领，深度挖掘党建数据，实时采集应用系统实时产生的行为数据，把日常的工作流程转化为计算机可以识别和分析的数据，将大数据、云计算、光传输等现代信息网络技术引入到基层党建工作中，引入精细化管理理念，建立智慧党建综合信息平台。

    互金科技智慧党建云平台不但能从海量的党务政务大数据中精炼最有价值的信息，同时通过可视化图表展示党员管理、组织管理、学习教育情况、活动管理、纪检监察等数据，从而获得准确的党员画像信息和丰富的党务成果信息，让党建工作全面量化，为党建决策提供强大的数据支撑。

## 软件架构
采用PHP+Mysql架构
前端uniapp
后台fastadmin

Bower (可选,用于管理前端资源)
Less (可选,用于编辑less文件,如果你需要增改css样式,最好安装上)

## 产品介绍

### 党组织管理
党组织信息填写
在对应党组织下填写党组织基本信息，包括党组织名称、党组织类型、上级党组织、党员总人数、建立时间、最后一次换届选举时间等信息。每一次信息变更，系统会形成一条变更记录。
对于党组织书记，可以填写姓名、职务、职级等信息。
对于党建联系人，可以填写姓名、联系电话、职务等信息。
对于活动情况，可以填写是否有专门活动场所、活动经费来源、活动经费数额等信息。
对于群团工作情况，可以填写是否建立工会、是否建立团组织、是否建立妇联等信息。
工作人员对党组织信息进行维护，同时还能够查看信息变更的记录，方便了解何时进行了哪些信息的修改。用户可以对党组织信息进行整体浏览。
党组织信息浏览
根据党组织关系，系统将以树形结构展示党组织各级结构。社会组织管理部门可查看各级党组织，点击某一党组织后可以显示该党组织基本信息、会议记录和党员信息。
### 党员管理
党员信息填写
系统支持对正式党员、预备党员、积极分子等人员进行相关的基本信息填写，包括姓名、性别、入党年月、现任职务、联系方式等信息。每一次信息变更，系统会形成一条变更记录。
党员信息浏览
系统能够统计现有党员数量，并进行展示，社会组织管理部门能够对所有党员名册进行浏览，并支持名册导出和打印功能。
党员转出
如有党员需要转出，社会组织在该党员信息下进行点击转出，则系统自动生成一条申请到社会组织管理部门，由社会组织管理部门进行审批。审批通过后，该党员信息会自动在党员名册中被删除。
党组织电子信息库管理
1.电子档案管理
对各级党员的人事档案进行分类、归档管理。
2.文件资料柜
为社会组织管理部门提供扫描纸质档案自动录入等文件资料管理服务，每条记录包括文档编号、文档名称、发布人、发布时间等信息。
3.数据导入
填写党员信息等相关内容后，将信息导入系统，系统自动生成主要组织架构及人员信息。
4.年度报表
将系统数据中党员信息按指定格式导出，自动生成上报组织的基础报表样式。
5.组织设置
设置社会组织管理部门基本信息。
6.人员设置
提供按党组织和社会组织管理部门两种查询人员的方式。可以查看修改人员信息可以添加人员。
7.流失党员汇总
运用LBS通过基站定时轮训党员位置，对党员流动轨迹进行记录，最终提供数据以供统计分析，汇总党员流失情况
8.电子通讯录
将所有党员的通讯地址信息统一录入，通过PC、手机终端两种呈现形式便于党员之间相互沟通。(将电子通讯录接入 PC\手机端)
### 发展党员管理
将发展过程细化为5个阶段、25 个环节、50多个节点对党员从申请党员 到成为正式党员全过程进行跟踪记录，对发展过程中的每一步进行有效规范。 包括流程设置节点预警时间限定、进度查询、历程追踪、工作提醒、文本查重、文件自动归档等功能。
### 组织生活管理
组织生活管理是对党组织日常生活事务的管理，组织生活主题、方式丰富多样，如何开展好、落实好组织生活及党员活动，对活动过程进行规范及有效监管，并取得良好成效是亟待解决的问题。
通过组织生活与党员活动管理模块实现对“三会一课”、 ”主题党日”、“特色党建"、”党建研究”等主题党建活动从计划到结束的各个环节的规范、记录、监管。
### 台账管理
系统针对基层党组织自动汇总、分类, 生成党建台账,方便党务工作者和领导查看党建工作成果,督促党建工作落实情况。党建台账包含组织信息、党员信息、组织生活、党员学习党建活动等。支持党建台账的导出及打印。
### 督察督办
为领导提供党建管理驾驶舱，各项工作进度、成果一目了然,对于工作开展滞后的党组织可进行一键督办，大大加强了监管的力度,监管手段更方便快捷。
### 党费缴纳
党费缴纳通过对接银行党费缴纳系统实现，用户在本系统党费缴纳板块可以直接跳转至银行党费缴纳界面，支持多种支付方式缴纳党费，党费由党员交至党组织，党组织统一进行缴纳，党费缴纳完成后银行可将缴纳情况反馈至本系统。
同时银行还能够将党费统一返还至党组织账户供党组织使用。
### 党费缴纳记录
根据银行反馈回来的党费缴纳情况，系统可以形成各社会组织党组织党费缴纳记录，包括党费合计金额、缴费年月等，管理人员可以进行查看浏览，对于还未缴纳党费的党组织，系统还能够进行预警提醒。
### 党费使用记录
系统支持各社会组织党组织记录党费使用情况，用户可以填写项目名称、负责人、使用金额、使用时间等情况，用户可以查询所有党费使用情况记录。
### 预警提醒
系统支持日程提醒功能，用户可以自定义添加党建活动、教育培训或其他重要事项安排的日程，在该日程前系统会进行弹窗提醒用户，确保用户不遗漏党建相关活动事项。
系统支持对党费收缴预警提醒、组织生活会会议纪要预警提醒、主题活动记录预警提醒、换届选举预警提醒，系统能够对以上事项在规定时限前对用户进行预警提醒，能够通过弹窗方式或短信方式进行提醒。
### 党内生活记录
用户可以发起组织生活会、主题活动等，并可对会议和活动信息进行填写，完成后可以上传相关资料，系统支持文档、图片格式上传。
### 三会一课
通过三会一课功能实现社会组织党组织在三会一课中会议方面的相关管理。
会议记录上传：会议结束后，社会组织在系统填写会议信息并上传会议纪要，系统支持文档和图片附件上传。
### 党员教育
党员教育信息填写
社会组织填写党员教育信息包括培训记录、志愿者队伍、在线学习、组织活动、党员表彰、纪律处分情况等，通过填写记录加强党员日常学习教育全程监督和管理。
党员教育信息汇总
系统会对每个社会组织的党员教育情况进行汇总统计，社会组织管理部门能够对党员教育情况（培训记录、志愿者队伍、在线学习、组织活动、党员表彰、纪律处分情况）进行浏览。
### 党组织成立审批
社会组织需要成立党组织时，通过此模块进行发起申请，填写详细申请信息，社会组织管理部门工作人员对此进行审批，审批通过后，党建管理的其他功能模块激活，社会组织才能进行使用，同时系统将提示社会组织线下提交纸质资料。
### 负责人审批
此功能仅涉及二党委，社会组织、党组织上传负责人相关需要审批的信息，由社会组织管理部门工作人员进行审批，审批完成后系统可生成函件，同时系统将提示社会组织线下提交纸质资料。
### 机关管理
机关管理：利用移动终端和移动通讯网络不受时间、空间限制的优越条件，为社会组织管理部门、社会组织党员提供随时、随地、随身的党务信息服务，有效管理社会组织党员基层工作。主要包括功能模块:会议通知、短信调研、信息发布、政策咨询、信息收集。
1、信息发布：信息发布功能可以在第一时间向各社会组织党员发布党的方针政策、文件精神和党建要闻。
2、短信调研：通过短信调研功能，可极大缩短调研时间，调研活动不仅快速、灵活，而且统计结果可支持导出。
3、信息收集：信息收集功能可以使各级组织部门及时了解社会组织党组织的思想动态，每时每刻倾听社会组织党组织的心声；社会组织党组织直接把政策反馈、政策咨询、意见建议、反映问题反馈到社会组织党组织，是“加强和改进新形势下党的建设”的重要体现。
4、信息互动：通过信息互动功能，各级组织部门可以为社会组织党员提供最权威、最准确、最快速的问题解答。通过双向互动的方式，为基层党员解决基层党组织建设、政策咨询等问题。
### 在线学习管理
（1）视频在线学习
按照“网络教学+互动答疑”的思路，在党建信息中开设远程课堂，实现远程互动教学管理。各级党组织可以不定期邀请有关专家到指定演播室讲课，实时传播到各社会组织党组织，并且可以进行实时互动交流，强化教学管理，满足社会组织党员群众的学习需求。
同时，利用网络视频可以开展社会组织党组织生活。党员登录平台后，可进入党组织预先设定的虚拟会议室，与其他党员面对面开展“两学一做”、民主评议党员等视频组织生活，还可以进行网上投票和举手表决等功能操作。对于时间安排不便的党员，社会组织党组织可将组织生活视频录制下来供党员点播观看，有效地解决了由于党员分散性和自主性强而难以集中开展社会组织党组织生活的问题。
视频在线学习由接入服务、视频播放服务、学习信息存储、播放客户端等四个部分组成。
1、接入服务:负责提供视频列表、判断客户端的类型及带宽，根据客户端信息指示视频播放服务器播放不同分辨率、不同码流的视频课件；
2、视频播放服务器:负责播放视频课件；
3、学习信息存储:在视频播放过程中，要将用户信息及播放信息存储到数据库中，作为学员学习的过程记录；
4、视频播放客户端，兼容win7及android3.5以上的版本。
（2）文档在线学习
文档学习服务有接入服务、文档转换服务、文档存储服务、学习信息存储及播放客户端组成。
1、接入服务:负责提供文档列表、判断客户端的类型，根据客户端信息指示文档转换服务播放不同分辨率的文档课件；
2、文档转换服务:从文档存储服务中提起用户要学习的文档，并转换成图片格式(jpg)，推送给用户；
3、在文档浏览过程中，要将用户信息及播放信息存储到数据库中，作为学员学习的过程记录；
4、文档浏览客户端，兼容windows xp、win7 及android2.2以上的版本；
5、文档在线学习的服务能力:支持1000-1500个并发访问。
（3）学习测评管理
党建平台学习人员通过PC/手机端进行:在线答题、学习小测、手机短信答题、综合测试等测评功能来考核学习情况。
### 统计分析
系统可以根据具体业务提供多张报表以供查询与分析，形成图表、柱状图、饼状图、折线图等多类型的统计分析表。包括：
1、党组织信息统计分析表
2、党员信息统计分析表
3、党费缴纳信息统计分析表
4、教育信息统计分析表等等
### 综合门户子系统
综合门户子系统新申请域名，面向省、市、县（市）区、乡镇（街道）、社区等社会组织的工作人员提供统一的处理各类日常业务工作的统一平台和入口。实现对门户上的栏目、文章进行新建、修改、删除、发布等基本功能；实现采购项目的展示；实现人才招聘的展示；实现培育计划的发布、删除、修改等管理功能。
